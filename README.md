## 项目简介

简单的在线商城, 暂未链接后端。

拥抱 `React` 函数式组件, `Hooks` 写法。

### 安装 & 运行

下载依赖

- `npm install`

运行本地服务器

- `npm run dev`

### 技术栈

- `React 17.0.2`

- `Vite` 脚手架

- `CSS Module`
